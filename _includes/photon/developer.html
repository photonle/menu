Debug Mode Reference

  • Toggle debug mode: photon_debug
    • Toggles build mode
  • Toggle debug mode: photon_debugtarget ELS/REG
    • Use 'ELS' for modifying emergency light positions
    • Use 'REG' for modifying regular light positions
    • This will 'target' the vehicle you are currently driving
  • Change light index: +/- keys on number pad
    • + will move index up
    • - will move index down
  • Copy vector/angle numbers: * on number pad
    • Will copy a valid Lua chunk for the Vector() and Angle() parameters of the light
  • Toggle vector/angle mode / on number pad
    • Alternates adjusting the vector or angle
  • Increase Increments Shift key
    • Increases adjustment intervals while held down
  • Decrease Increments Alt key
    • Decreases adjustment intervals while held down
  • Adjust Vector X/Angle Pitch Left/Right Arrow Keys
    • Remember this is always relative to the car, not how you're looking at it
  • Adjust Vector Y/Angle Yaw Up/Down Arrow Keys
    • Remember this is always relative to the car, not how you're looking at it
  • Adjust Vector Z/Angle Roll CTRL + Up/Down Arrow Keys
    • Remember this is always relative to the car, not how you're looking at it

Starting a Photon Car

An Introduction

Photon is designed to give developers (like yourself) the widest possible range of flexibility in your creations. Making cars for Photon invovles dealing with a big learning curve, but you have to trust there is a method to the madness, and it will eventually make sense.

Photon, above all, is designed to allow for creating any car setup you imagine. While its biggest 'selling-point' is the emergency lighting, it also has growing support for general car lighting to add a touch of realism to your server or scene. The trade-off with flexibility is ease of use. While you can create virtually any kind of lighting setup, it takes patience, practice and precision.

For all intents and purposes, Photon is still in its infancy. Features are modified and changed all the time based on community response and evolving Garry's Mod functions. While the structure is unlikely to change for a very long time, more tools will be released to make this process easier. It's important to note that you do not need coding or Lua experience to make a vehicle. What you're creating is data -- data that Photon processes to create the effects. The only requirement is a basic understand of how everything relates.
