Official Component Documentation

Federal Signal AeroDynic LED
Lightbar commonly used on FDNY engines. Has 24 LEDs and 2 rotators.

EMV.Auto code:
{
 ID = "Federal Signal AeroDynic FDNY New",
 Scale = 1,
 Pos = Vector( 0, 0, 0 ),
 Ang = Angle( 0, 0, 0 )
},

Additonal EMV.Auto options: (only needed if changing light colors)
 Color1 = "RED",
 Color2 = "WHITE"


Federal Signal Vision SLR Split
Lightbar similar to the Vision SLR but larger. Has 11 LED rotators. Takedown illumination compatible.

EMV.Auto code:

FDNY Rear Engine Lightbar
Rear lightbar commonly seen on FDNY engines in pairs. Inlcudes both new (no phase and A) and old (phases B and C) flash patterns.

EMV.Auto code:
{
 ID = "FDNY Rear Lightbar",
 Scale = 1,
 Pos = Vector( 0, 0, 0 ),
 Ang = Angle( 0, 0, 0 )
},

Additonal EMV.Auto options:
 Color1 = "RED",
 Color2 = "AMBER",
 Color3 = "BLUE",
 Phase = "A",
 Phase = "B",
 Phase = "C"
