About Photon Fire & Rescue

About

Photon Fire & Rescue is an official addon to the Photon Lighting Engine. It allows for the addition of new lighting and features specifically targeted to increasing the accuracy of fire and EMS vehicles. This addon requires Photon to operate as it simply builds upon the already existing engine.

How can I add Photon Fire & Rescue to my server?

If required, a workshop file may link to the addon in order to operate properly. If it does not, then Photon Fire & Rescue is not required to use Photon. F&R can be added via your server's Workshop collection and is strongly recommended in order to stay up to date. If you wish to add custom or unofficial Photon vehicles to your server, you must add them separately, along with any other requirements.

Can I request a certain engines or x-car be added?

At this time there are a few Photon Official engines in the works. As for engines/EMS vehicles on the workshop now these can be updated to include the new lighting if their creator wishes.

Official Component Documentation

Federal Signal AeroDynic LED
Lightbar commonly used on FDNY engines. Has 24 LEDs and 2 rotators.

EMV.Auto code:
{
 ID = "Federal Signal AeroDynic FDNY New",
 Scale = 1,
 Pos = Vector( 0, 0, 0 ),
 Ang = Angle( 0, 0, 0 )
},

Additonal EMV.Auto options: (only needed if changing light colors)
 Color1 = "RED",
 Color2 = "WHITE"


Federal Signal Vision SLR Split
Lightbar similar to the Vision SLR but larger. Has 11 LED rotators. Takedown illumination compatible.

 Color2 = "AMBER",

 Phase = "A",
 Phase = "B",
 Phase = "C"
